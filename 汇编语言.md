## 1.2

内存大小由地址总线宽度决定

N 位的机子 ⟹ 地址总线宽度 N⟹$2^NB$

<center>

|  N  |      支持的内存大小      |
| :-: | :----------------------: |
|  8  |    $\rm 2^{8}B=256B$     |
| 16  | $\rm 2^{16}B=2^6KB=64MB$ |
| 32  | $\rm 2^{32}B=2^2GB=4GB$  |
| 64  |  $\rm 2^{64}B=2^{34}GB$  |

</center>
